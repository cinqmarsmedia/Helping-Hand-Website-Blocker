Helping Hand
A non-profit web blocker 1.0.0
This tool is designed for anyone who wants to remove the temptation of certain web habits, parents who wish to control the web accessibility of their child's computer, or for anyone who wants a GUI to blocklist certain domains. Blocking will continue after the program is quit but can be easily removed using the "Revert + Undo" menu above with the provided password.

Browse from some of our recommended presets above or use in custom mode to add your own sites to blacklist. This software is also open-source and available on GitHub under the MIT License. It is made available as is by Cinq-Mars Media, an educational and tech non-profit.
